# Enterprise Library Management System

A modern full-stack library management platform built with **Next.js**, **TypeScript**, **PostgreSQL**, **Redis**, and **Auth.js**. The application enables administrators and users to efficiently manage library operations through a secure, scalable, and intuitive interface.

---

## Features

### Authentication & Authorization
- Secure authentication with Auth.js
- Role-based access control (Admin/User)
- Protected routes
- Session management

### Admin Dashboard
- Dashboard with key library statistics
- Manage books and inventory
- Manage users
- Search and filter records
- Track borrowing activities

### Library Management
- Add, update, and delete books
- Book availability tracking
- Borrow and return books
- View borrowing history
- Book categorization

### Performance
- Redis caching for faster data access
- Optimized database queries
- Server-side rendering with Next.js
- Responsive UI
